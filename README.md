# _19 ASP.NET Core Login & Register Practice

A simple ASP.NET Core MVC authentication practice project built using ASP.NET Core Identity. This project demonstrates the complete user registration and login workflow with server-side validation and a clean service-layer architecture.

---

## Features

* User Registration
* User Login
* Confirm Password Validation
* Email Availability Check
* Password Hashing with ASP.NET Core Identity
* Authentication using UserManager and SignInManager
* Model Validation
* Service Layer Architecture
* Success Alert Messages using TempData
* Bootstrap Responsive UI

---

## Technologies Used

* ASP.NET Core MVC
* ASP.NET Core Identity
* Entity Framework Core
* SQL Server
* Bootstrap

---

## Learning Outcomes

* Implement user registration and login
* Validate user input using Data Annotations
* Prevent duplicate email registration
* Verify password confirmation during registration
* Authenticate users securely with Identity
* Display user-friendly success messages with TempData
* Understand the authentication flow using UserManager and SignInManager
